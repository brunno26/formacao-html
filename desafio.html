Tags h1 - h6

[Voltar]

Essas tags são elementos do HTML que são utilizados para definir títulos e subtítulos.

Elas são utilizadas para indicar a hierarquia dos títulos.

Sendo h1 para a mais relevante e h6 para a menos relevante.

Uso da Tag p

[Voltar]

É utilizada para definir um parágrafo de texto.

Ela cria um espaçamento antes e depois do conteúdo para separar os parágrafos.

Tag blockquote

[Voltar]

Usada para exibir as citações em bloco.

Nota-se que ao ser utilizada ela cria um recuo visual para destacar o conteúdo.

Tag i

[Voltar]

Muito simples, essa tag é usada para deixar os textos em itálico.

Tag strong

[Voltar]

Essa tag é utilizada para deixar os textos em negrito

Tag u

[Voltar]

Essa tag é usada para deixar os textos sublinhados, conhecida como underline.

Tag sup

[Voltar]

A tag sup é utilizada para deixar uma letra elevada.

Lista Ordenada

[Voltar]

Como o próprio nome diz, cria uma lista ordenada.

E essa lista possui uma sequência número como: 1, 2, 3.

Lista Não Ordenada

[Voltar]

O próprio nome sugere que a lista não segue uma ordem.

Diferente da ordenada os tópicos não possuem sequência.

Links

[Voltar]

Usados para criar conexões "links" para outras páginas
